# 🌸 Genki Dojo（元気道場）

A calm, modern Japanese learning app inspired by **Genki I**.
No sign-up, no downloads — just open and start studying.

👉 **Live app:** https://genki-dojo.netlify.app/

---

## ✨ Overview

Genki Dojo is a lightweight, browser-based study app that brings together:

* lessons
* grammar
* vocabulary
* kana practice
* speaking

All in one clean, distraction-free experience.

---

## 📱 Features

### 🈳 Kana Practice

* Hiragana (あ → ん)
* Katakana (ア → ン)
* Tap-to-learn + flashcards
* Progress tracking

---

### 📖 Lessons

* Structured around Genki I
* Grammar explanations + examples
* Easy navigation between topics

---

### 🧠 Vocabulary

* Flashcard-based learning
* Mark words as learned
* Quick reference lists

---

### ✏️ Exercises

* Multiple choice
* Sentence writing
* Sentence completion

---

### 🎙 Speaking Practice

* Browser-based speech recognition
* Compare pronunciation with target phrases

> Best supported in Google Chrome

---

### 🗾 Survival Japanese

* Real-world phrases
* Travel-friendly categories

---

## 📲 Install as an App (PWA)

Genki Dojo can be installed like a mobile app:

* Open the site in Chrome or Safari
* Tap **“Install”** or **“Add to Home Screen”**
* Launch it like a native app

Works offline after installation.

---

## 🚀 Run Locally

```bash
git clone https://github.com/YOUR-USERNAME/genki-dojo.git
cd genki-dojo
open index.html
```

---

## 📁 Project Structure

```text
index.html
manifest.webmanifest
sw.js
icons/
```

* `index.html` → main app
* `manifest.webmanifest` → app install settings
* `sw.js` → offline support
* `icons/` → app icons
